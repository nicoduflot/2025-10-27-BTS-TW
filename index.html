Reboot CSS

La première partie du CSS cde bootstrap est un "reboot", c'est un dire toute une partie CSS qui permet de remettre en mode web et accessible les éléments normalement couvert par le CSS du navigateur, qui n'est pas accessible par défaut.

La police de caratère est modifiée, passant d'une police à empatement non adaptée à l'affichage à l'écran par une police sans empatement. La taille de la police par défaut est augmentée ainsi que l'interligne.

Tous les éléments sont par défaut mis en calcul de taille avec les bordures, alors que le CSS navigateur gère les tailles des éléments par leur contenu. De plus, la marge extérieure de body est supprimée.

Les tableaux

Le reboot css de Bootstrap règle les problèmes d'affichage des tableaux dûs au CSS du navigateur. En ajoutant la classe "table", la présentation de base est améliorée.

Il est possible de changer les couleurs, l'apparence, et d'ajouter des bordures aux tableaux en utilisant les classes Bootstrap

Tout comme les couleurs, les mots clefs "primary", "secondary", "dark", "success", "danger", etc. permettront d'influer sur les couleurs de fond des tableaux.

Il est possible d'agir au niveau du tableau en entier, sur une ligne ou sur une cellule.

Différentes classes sur les différentes parties du tableau
La ligne est en warning La ligne <tr> étant en warning, <th> et <td> sont en warning Cette <td> à la classe "table-primary" qui surcharge l'héritage de la classe apposée à <tr>

Il est possible d'appliquer une couleur de fond qui alterne une ligne sur deux avec la classe "table-striped". Les couleurs choisie seront en relation avec la couleur apposée au tableau en entier.

Les formulaires

Les formulaires comportent toute une liste de classes qui permettent de mettre facilement en forme les champs de saisie.
De plus, Bootstrap permet de customiser les contrôles de formulaires, et cela automatiquement grâce au JS, et à l'utilisation des classes adéquates.
