Créer son CSS à l'aide de Sass

Des variables

Sass utilise des variables, il faut les écrire :
$code-color: #d63384;

Et quand on écrit dans le fichier Sass : 

code{
    color: $code-color;
}

et le css suivant est généré : 

code {
    color: #d63384;
}

Des Listes

Il est possible d'ajouter plusieurs éléments dans une variable, il s'agit d'une liste (on peut comparer ça un tableau mais il n'y a pas d'indice des éléments de la liste) 

$breakpointsList: 540px, 720px, 690px;

On peut donc faire une boucle sur les éléments de la liste 

@each $breakpoint in $breakpointsList{
    .maxwidth#{$breakpoint}{
        max-width: #{$breakpoint};
    }
}

Et cela génère le css suivant : 

.maxwidth540px {
    max-width: 540px;
}
    
.maxwidth720px {
    max-width: 720px;
}
    
.maxwidth690px {
    max-width: 690px;
}

Des "tableaux"

En Sass on peut utiliser ce qu'on appelle des "map", il s'agit de tableaux dont l'indice est une clefs au lieu d'un indice.

ces Tableaux permettent de générer des éléments récurrents mais différents selon les valeurs et les clefs du tableau.

Par exemple, on crée deux map pour gérer un affichage responsive d'un conteneur 

// map, il faut ajoute sass:use map en début de script
$breakpoints: (
  sm: 576px,
  md: 768px,
  lg: 992px,
  xl: 1200px,
  xxl: 1400px
);

$rwd-max-width: (
  sm: 540,
  md: 720,
  lg: 960,
  xl: 1140,
  xxl: 1320
);

On peut ensuite utiliser des maps pour faire des boucles plus complètes : 

.container{
    margin-left: auto;
    margin-right: auto;
    padding-left: $gutter-x;
    padding-right: $gutter-x;
}

@each $breakpoint, $value in $breakpoints {
    @media screen and (min-width: #{$value}) {
        .container{
            max-width: #{map.get($rwd-max-width, $breakpoint)}px;
        }

        .row > *{
            max-width: #{map.get($rwd-bp-column-width, $breakpoint)};
        }
    }
}

On obtient ainsi le css suivant : 

@media screen and (min-width: 576px) {
    .container {
        max-width: 540px;
    }
}
@media screen and (min-width: 768px) {
    .container {
        max-width: 720px;
    }
}
@media screen and (min-width: 992px) {
    .container {
        max-width: 960px;
    }
}
@media screen and (min-width: 1200px) {
    .container {
        max-width: 1140px;
    }
}
@media screen and (min-width: 1400px) {
    .container {
        max-width: 1320px;
    }
}

Les "mixins"

Les mixins permettent de facilement écrire les dérivations d'une classe basique. On décrit un comportement, que l'on ajoute à une classe par exemple, mais ce comportement contient une variables par défaut.

On peut ensuite écrire une autre classe qui appelle ce comportement mais on remplace la valeur de la variables par défaut. 

@mixin alert($alert: lightblue){
    background: $alert;
    box-shadow: 0 0 1px rgba($alert, 0.25);
    color: black;
    margin: $gutter-y;
    padding: $gutter-x;
    border-radius: calc($gutter-x/2);
}

On écrit ensuite la classe principale qui appelle généralement la mixin seulement ainsi que les dérivation ou on remplace la variable de couleur de fond. 

.info{
    @include alert;
}

.danger{
    @include alert($alert: orange);
}

.success{
    @include alert($alert: lightgreen);
}

Une fois compilé, on obtien le css suivant 

.info {
    background: lightblue;
    box-shadow: 0 0 1px rgba(173, 216, 230, 0.25);
    color: black;
    margin: 0.5rem;
  padding: 0.5rem;
  border-radius: 0.25rem;
}
    
.danger {
    background: orange;
    box-shadow: 0 0 1px rgba(255, 165, 0, 0.25);
    color: black;
    margin: 0.5rem;
    padding: 0.5rem;
    border-radius: 0.25rem;
}
    
.success {
    background: lightgreen;
    box-shadow: 0 0 1px rgba(144, 238, 144, 0.25);
    color: black;
    margin: 0.5rem;
    padding: 0.5rem;
    border-radius: 0.25rem;
}

Le "nesting"

Le nesting c'est le principe d'encapsulation du CSS. Il et compiler pour écrire simplement du css basé par exemple sur la structure du HTML

Par exemple : 

@mixin links($color: darkblue, $hover: darkred){
    a{
        color: $color;
        &:hover{
            color: $hover;
        }
    }
}
body {
    margin: 0;
    background-color: $body-background;
    font-family: $body-font;
    color: $body-color;
    font-size: $body-size;
    a {
        text-decoration: none;
    }
    > {
        nav{
            ul{
                list-style-type: none;
                margin: 0;
                > li{
                    @include links;
                }
            }
        }
        footer{
            nav{
                ul{
                    li{
                        @include links($color: darkred, $hover: darkblue);
                    }
                }
            }
            @include links($color: darkred, $hover: darkblue);
        }
    }
}

Compile le css suivant : 

body {
    margin: 0;
    background-color: #f4f5f0;
    font-family: Arial, Verdana, Geneva, Tahoma, sans-serif;
    color: #333;
    font-size: 16px;
}
body a {
    text-decoration: none;
}
body > nav ul {
    list-style-type: none;
    margin: 0;
}
body > nav ul > li a {
    color: darkblue;
}
body > nav ul > li a:hover {
    color: darkred;
}
body > footer nav ul li a {
    color: darkred;
}
body > footer nav ul li a:hover {
    color: darkblue;
}
body > footer a {
    color: darkred;
}
body > footer a:hover {
    color: darkblue;
}
