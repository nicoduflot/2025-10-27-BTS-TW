Créer des composants avec Tailwind CSS

Les composants

@layer @apply

Les composants permettent de créer des éléments réutilisables plustôt que d'écrire à chaque élément récurent la même suite de classes. 

@layer components {
  .btn {
    @apply px-4 py-2 rounded font-semibold transition duration-200;
  }

  .btn-primary {
    background-color: var(--color-blue-500);
    color: var(--color-white);
    &:hover {
      @media (hover: hover) {
        background-color: var(--color-blue-700);
      }
    }
  }
}
